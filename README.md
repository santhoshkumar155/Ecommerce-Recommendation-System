# E-commerce Recommendation System

This project implements an E-commerce Product Recommendation System using Machine Learning.

🔹 Features:
- Recommends products based on user ratings
- Uses Collaborative Filtering (SVD)
- Built with Python and Streamlit UI

🔹 Technologies Used:
- Python
- Pandas
- Surprise Library
- Streamlit

🔹 How it works:
The system predicts user preferences based on past ratings and recommends top products.

🔹 Output:
- Displays top recommended products for selected users
